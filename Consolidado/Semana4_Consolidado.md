[Ver el reporte completo](../Actividad3.md)

Se desarolló un modelo de regresión lineal simple para predecir la relación entre el número de carreras y el número de bateos, utilizando un dataset real de ESPN. 

Se realizó la preparación de los datos, el análisis de correlación y la construcción del modelo, el cual fue evaluado con varias métricas que nos ayudaron a comprobar una relación positiva entre las variables, pero un bajo desempeño predictivo del modelo. 

Finalmente a través de las visualizaciones, se observo que el modelo tiende a concentrar las prediciones en un rango reducido, lo que sugiere que una sola variable independiente no es suficiente para explicar completamente la variabilidad de las carreras.
